## Note

Selection window: 2026-07-20T13:37 UTC through 2026-07-21T13:37 UTC. All eight WSJ Dow Jones RSS feeds, all ten NYT section RSS feeds, and all eight NBC section RSS feeds listed in the source instructions returned HTTP 200 and were used to build each outlet's candidate pool; items were filtered on `pubDate`/`dc:date`, not on feed-retrieval time. AP does not publish a public RSS feed, so its candidates were sourced via Google News's `site:apnews.com when:1d` search feed. The `news.google.com/rss/articles/...` redirect links in that feed no longer resolve through a simple HTTP redirect or through headless-browser navigation (confirmed by testing both approaches), so each AP candidate headline was instead cross-referenced against apnews.com's own topic-hub pages (business, world, Israel-Hamas war, United Kingdom, Donald Trump) to locate the matching canonical `apnews.com/article/...` URL, which was then fetched directly and confirmed via its `headline` and `datePublished` metadata to verify both exact wording and that original publication fell inside the 24-hour window. One NBC item and one WSJ/NYT/AP-adjacent story point to `today.com` video pages because no text-article equivalent existed in NBC's own feeds at fetch time; WSJ and NYT article pages return bot-protection HTTP status codes (401/403) to automated requests without a browser session, so those five links could not be live-verified by fetching the page directly — they are the exact, unmodified `<link>` values published in each outlet's own official RSS feed.
