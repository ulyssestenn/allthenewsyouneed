Selection window: 2026-07-21T13:37 UTC through 2026-07-22T13:37 UTC. All eight WSJ Dow Jones RSS feeds, all ten NYT section RSS feeds, and all eight NBC section RSS feeds listed in the source instructions returned HTTP 200 and were used to build each outlet's candidate pool; items were filtered on `pubDate`/`dc:date`, not on feed-retrieval time. AP does not publish a public RSS feed, so its candidates were sourced via Google News's `site:apnews.com when:1d` search feed, which also returned HTTP 200. That feed's `news.google.com/rss/articles/...` links redirect back into Google's own SPA shell rather than to apnews.com, so they could not be resolved directly (confirmed via curl `-D -` header inspection and via the WebFetch tool). Each AP candidate was instead cross-referenced against apnews.com's own topic-hub pages (Iran, Donald Trump, Israel-Hamas War, Saudi Arabia, Congress), fetched with a browser user agent, to locate the matching canonical `apnews.com/article/...` URL; each candidate URL was then fetched directly and its exact headline and publication time verified against the page's `og:title` and `datePublished` metadata before inclusion. WSJ and NYT article pages return bot-protection HTTP status codes to automated requests without a browser session, so those links could not be live-verified by fetching the page directly — they are the exact, unmodified `<link>` values published in each outlet's own official RSS feed, with WSJ's feed-tracking query parameters (`?mod=...`) stripped for a clean canonical URL.
